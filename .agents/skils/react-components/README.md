# React Components Skill

This skill creates React components following Qwantic Coders project conventions.

## Usage

When you need to create a new React component, use this skill. It will:
1. Create the component with proper TypeScript typing
2. Use CSS for styling
3. Follow the project's naming conventions

## Example

See `examples/Button.tsx` for a reference component.

## Structure

```
react-components/
├── SKILL.md              # Skill definition
├── examples/
│   ├── Button.tsx        # Example component
│   └── Button.css # Example styles
└── README.md             # This file
```

## Key Conventions

- Use `FC` type for functional components
- Props interface: `Props`
- CSS classes in kebab-case
- Export both named and default
